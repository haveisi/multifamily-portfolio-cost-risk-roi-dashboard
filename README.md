\# Multifamily Portfolio Cost, Risk \& Retrofit ROI Dashboard



This project is a property-management and asset-planning dashboard for a simulated multifamily real estate portfolio. It connects building performance, utility cost, emissions exposure, waste efficiency, and retrofit financial analysis into one decision-support workflow.



The goal is to help property managers, asset owners, and sustainability teams identify which buildings need attention first, which operating issues are driving cost and risk, and which retrofit actions have the strongest payback and estimated value impact.



\## Dashboard Pages



1\. Portfolio Performance \& Risk Overview

2\. Utility Performance \& Operating Cost Risk

3\. Capital Planning \& Retrofit ROI Priorities



\## Key Questions



\- Which properties are costing more to operate?

\- Which buildings show higher energy-use and carbon-risk patterns?

\- Which properties should be prioritized for operational review?

\- Which retrofit actions offer the strongest payback?

\- How could utility savings affect estimated property value?



\## Key Metrics



\- Energy use intensity

\- Utility cost per unit

\- Scope 1 and Scope 2 emissions

\- Carbon intensity

\- Waste diversion rate

\- Integrated property risk score

\- Retrofit net cost

\- Annual savings

\- Simple payback

\- Estimated value impact



\## Tools Used



\- Excel for data modeling, formulas, and calculation guide

\- Power BI for dashboard design and visual analytics

\- DAX for portfolio KPIs and financial measures

\- Python for project organization and optional QA

\- Real estate performance analytics

\## Important Note



This project uses simulated and screening-level portfolio data for analytics demonstration. It is not an official GRESB score, investment-grade audit, or engineering-grade retrofit assessment.



\## Portfolio Value



The value of this project is not only tracking sustainability metrics. It shows how property performance data can support real operating and capital-planning decisions by connecting utility cost, building performance, risk, retrofit payback, and estimated value impact.
